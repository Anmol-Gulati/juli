Say hello to Juli!

Juli helps you run web apps like real Mac apps with dock badge, menu bar integration & more.

 Download for macOS

Version {{ site.juli_version }}. Requirements: macOS 10.9+.

Websites like Gmail, Facebook Messenger and Instagram are becoming more and more like desktop applications. But running each of these sites in a separate tab in your browser can be a real pain. It not only reduces your productivity but also invades your privacy with cross-website trackers.

Juli lets you create a desktop app (site-specific browser) out of any website, helping you to work more efficiently & protect your privacy (each app runs within a sandboxed container).

Juli is optimized for macOS with many integrations like notification, dock badge and menu bar support. Compared to its sibling, WebCatalog Lite, Juli relies on Electron instead of Chromium and offers more features and customizations.

Juli icon is created from Aquarius icon by Bernar Novaly.
